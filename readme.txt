== Description ==

This is awesome capital letter plugin. This plugin make nice capital letter before every paragraph. This is help to attract to users.

Committable with all browsers. 

Features:
1. Awesome Style.
2. All browser support.
3. Nice theme option.
4. Unlimited Color. 
5. You can change font style.

You can see demo here <a href="http://facebond.com/plugins/rkp-capital-letter/" target="_blank">DEMO</a>

If you like this plugin please leave your ratings.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
